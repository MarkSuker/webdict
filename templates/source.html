Requesting more dictionaries

If you would like a new dictionary added, feel free to create an issue for it on GitHub! Make sure to include the URL to the dictionary content, ideally a compressed file, or a database that can be queried easily to extract the content. Also make sure the content has a permissive license that allows free distribution online.
